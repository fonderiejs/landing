# Fonderie branding

- `src/` — HTML/SVG sources; render with headless Chrome:
  `chrome --headless --screenshot=export/x-banner.png --window-size=1500,500 src/banner.html`
- `export/` — ready-to-upload assets (X banner 1500×500, X avatar 800×800)
